# OpenClawが「本格的なAIエージェント」になる3つの核となる仕組み

OpenClawはただのチャットAIではありません。  
あなたのPC/Mac/Raspberry Piで**24時間365日動き続け**、実際にメールを送ったり、ファイルを整理したり、コードを書いたり、**自分で判断して行動**する「デジタル分身」のような存在です。

その本質を支えるのが、以下の**3つの仕組み**です：

### 1. スキルの追加（Skills）  
→ 「アプリをインストールするように、どんどん新しい能力を追加できる」

**どんな仕組み？**  
- スキルは**Markdownファイル（SKILL.md）**だけで作られています。  
  中身は「このスキルは何をするか」「どうやって使うか」「必要なAPIキーやツールは？」という説明だけ。
- **ClawHub**（公式スキルストア）から  
  `clawhub install gmail`  
  という1コマンドで即インストール完了！  
  （現在3000個以上ものコミュニティスキルが存在）
- インストールされたスキルは**自動で読み込まれ**、エージェントの「使えるツール一覧」に追加されます。  
  しかも**ホットリロード**（ファイルを編集するだけで即反映・再起動不要）。

**具体例**  
- Gmailスキル → 「今日の未読メール全部要約して」  
- ブラウザ自動操作スキル → 「競合サイトの価格を調べて表にまとめて」  
- Zenn自動投稿スキル → 「今日の学びを記事に書いてGitHubにPR作って投稿までやって」

**なぜ本格的？**  
普通のAIは「できること」が最初から決まっていますが、OpenClawは**無限に拡張可能**。  
あなたが「この作業を自動化したい」と思ったら、スキルを探すか、自分で（またはAIに作らせて）追加するだけ。  
→ これで「ただ答えるAI」から「実際に手を動かすAI」になります。

### 2. メモリの永続化（Persistent Memory）  
→ 「一度覚えたことは絶対に忘れない。第二の脳になる」

**どんな仕組み？**  
すべて**あなたのPC内のMarkdownファイル**として保存されます（データベース不要）。

主なファイル（`~/clawd/` または `~/.openclaw/` フォルダ内）：
- **SOUL.md** → エージェントの「人格・価値観・あなたへの理解」（「私は丁寧で効率重視」「ユーザーは東京在住のエンジニア」など）
- **MEMORY.md** → 長期記憶（あなたが大事にしたいこと、過去の決定、好みなど）
- 日次ログ（`2026-02-25-conversation.md` など） → その日の全会話・行動履歴
- AGENTS.md / TOOLS.md → 現在の設定と使えるスキル一覧

**どうやって永続化される？**  
- メッセージが来るたびに、エージェントは**ディスクから全記憶を読み込んで**判断します。
- 重要なことは自動で**MEMORY.mdにまとめ直して保存**。
- PCを再起動しても、電源を落としても、**すべて残っています**。
- あなたがテキストエディタで直接編集・Git管理も可能！

**実感できる例**  
- 1週間前の「来月の旅行は京都希望」と話したら、今日「旅行の予定教えて」と言っただけで即思い出して「新幹線予約しますか？」と提案。
- 「もうChatGPTには戻れない」と言われる最大の理由です。

**なぜ本格的？**  
普通のAIは会話が終わると記憶がリセットされますが、OpenClawは**あなたと一緒に育つ**。  
使えば使うほど「あなた専用」のAIになっていく。これが「本物のエージェント感」の正体です。

### 3. マルチエージェント（Multi-Agent）  
→ 「1人で全部やるんじゃなく、専門のAIチームを作れる」

**どんな仕組み？**  
1つのOpenClaw（Gateway）の中に、**複数の独立したエージェント**を同時に動かせます。

各エージェントは：
- 独自の**ワークスペース**（フォルダ）
- 独自の**SOUL.md / MEMORY.md**
- 独自の**スキルセット**
- 独自の**モデル**（Claude 3.5 Sonnetは仕事用、速いモデルは雑用用）

**設定の仕方（簡単）**  
```bash
openclaw agents add coding     # プログラミング専門エージェント作成
openclaw agents add personal   # プライベート用エージェント作成
```

**メッセージの振り分け**も自動：
- 仕事のSlack → codingエージェント
- 家族のWhatsApp → personalエージェント
- 「@research 〇〇について調べて」→ researchエージェント

**さらにすごい点**  
- エージェント同士で**直接会話**できる（agentToAgentツール）
- 共有メモリも設定可能（チームで同じMEMORY.mdを見る）
- 1台のMac miniで**10個以上の専門AI**を24時間動かしている人もいます！

**なぜ本格的？**  
人間の会社と同じく「役割分担」ができるから、複雑な仕事もこなせます。  
「開発エージェントがコード書いて、レビューエージェントがチェックして、運用エージェントがデプロイ」みたいな流れが自然に実現。

### まとめ：この3つが揃うとどうなるか
| 仕組み           | 普通のAI          | OpenClawの本格さ                     |
|------------------|-------------------|-------------------------------------|
| スキルの追加     | 固定機能          | 無限拡張（アプリみたい）            |
| メモリの永続化   | 会話ごとに忘れる  | 一生覚えている第二の脳              |
| マルチエージェント | 1人で全部         | AIチームで分業・協力                |

これらが**全部ローカル・ファイルベース・オープンソース**で動くから、  
「自分のPCで自分のためにだけ動く、本物の自律型AIエージェント」になるんです。

実際に多くの人が「人生が変わった」「もうこれなしでは生きられない」と言っています。
