## 主題

ネットのイノベーターやアーリーアダプターの間で大流行しているOpenClawだが、リスクもあって、導入に迷う人も多い。
安全にかつコストパフォーマンスよくまた使い勝手よくOpenClawを導入したらどうしたらいいか？
それについて紹介する。

3. オープンクロー導入の現実的な選択肢とそのメリットデメリット（ここがメイン）
  - 使ってない古いPCがあって、メモリが8GB以上あるなら、それを使うのもよし。ノートPCでもOK。普段は蓋を閉じたまま運用できる。
  - 新たに安く導入する場合、現在話題になっているのは以下の4つだ。中古のM1 Mac mini、Windows ミニPC、Raspberry Pi 5、クラウドVPSだ。
  - 今回は初期コスト5万円以下で入手できるもの、という観点で比較してみた。
  - M1 Mac miniはCPUパワーの高さと、省電力性、静音性、省スペース性、そしてネイティブにOpenClawが動く点でベストマッチだが、メモリが固定で拡張できない。ストレージを増やすときもUSB 3で外付けするしかない。世界でも日本でも人気で中古品は入手しづらくなっており、より高性能だが値段が張るM2 Mac Miniに関心が移りつつある。
  - 5万円以下のWindows ミニPCは、CPUにインテルのN100を搭載したものが2.5万円から3.5万円。より強力なAMD Ryzen 5 7430Uを搭載したものが3.2万円から5万円。メモリ16GBで、DIMMの換装で64GBまで拡張できるものもある。CPUパワーではM1 Mac miniに対して、N100が約50%、Ryzen 5 7430Uが75%程度。Windowsで動かす場合はWSLというマイクロソフト公式Linux互換機能を使うのだが、CPUやメモリの効率でオーバーヘッドがある。Webブラウザの自動運転は処理が重いのでRyzen 5のほうがオススメだ。
  - Raspberry Pi 5は低価格、省電力、コンパクトだが、CPU性能はM1 Mac Miniの35%くらいとさらに下がる。価格面ではWindows ミニPCとそれほど変わらない。OSはLinuxなので、Linuxに慣れている人には使いやすいだろう。
  - クラウドVPSとはいわゆるレンタルサーバーのことだ500円から1000円程度のサービスは多くあるが、多くはメモリが2GBや4GB程度になる。「Contabo Cloud VPS 10」は月額6.85ユーロでリーズナブル。ただし東京リージョンは9.20ユーロ（約1500円）となる（ネットワーク的に遠いと反応に時間がかかる）。メモリ24GBを完全無料で借りられる「Oracle Cloud」の「Always Free」もあるが、競争が厳しくてなかなか借りられない、使用率が低いと回収されてしまうなどリスクが多い。
  - （CPUベンチマーク表、Mac Mini中古価格表、Windows ミニPC価格表、Raspberry Pi 5 スターターキット価格表、クラウドVPS価格表をそれぞれ適所に入れる）
  4. 筆者の場合は、Ryzen 5 7430U搭載の「ACEMAGIC Kron Mini K1」を導入してみた。Amazonで4万5998円だった。数年ぶりのWindowsマシン導入だったが、Windowsのセキュリティ設定がガチガチすぎて自分のマイクロソフトアカウントにログインできない、JISキーボードを認識しないなどの設定にかなり時間を取られたが、WSLの導入は問題なかった。OpenClawは運悪くバグありバージョンに当たってしまい、相談したChatGPTも迷走して1日潰れた。相談相手をClaudeに切り替えてOpenClawのバグと判明し解決したが、オープンソースソフトウェアはこういうこともある。現在はコンテンツの完全自動生成や2次元アイドル的アシスタントなどの実験を行っている。
  5. 電気代を含む3年間の総所有コストで比較すると、N100系のWindows ミニPCのコスパが良いようだ。イニシャルコストを抑えてすぐ体験したいならVPSを借りてやってみて、やれること、やりたいことが決まったら、ミニPCやMac miniの導入を考えるのが良いかもしれない。なお、OpenClawの運用には別にLLMの運用費用もかかる。OpenAIのChatGPTのサブスクを契約していれば、それも利用できる。最近は中華系LLMの性能向上が著しく値段も安い。OpenClaw界隈では、Kimi K2.5、MiniMax M2.5、DeepSeek V3.2などの人気が高く、アリババのQwen3はOpenClaw用の無料枠も用意されている。用途によっては選択肢に入れてもよいだろう。
